# Stardict java package
A package to read stardict dictionaries, without needing to uncompress dz files, with support for syn files.

Get the package from maven [here](https://search.maven.org/#search%7Cga%7C1%7Ca%3A%22StarDict%22) .

# Software developer instructions
Use intellij to push to maven.
